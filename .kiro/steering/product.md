# Product Overview

This is a workflow automation and agent management platform with a focus on social media automation and AI-powered assistance.

## Key Features

- **Workflow Builder**: Visual workflow creation and management using React Flow
- **Maya Agent**: AI agent interface for task automation and phone-based interactions
- **Social Media Automation**: TikTok and LinkBio automation capabilities
- **Task Management**: Task tracking and trigger-based automation
- **Dashboard**: Central hub for monitoring and managing automations

## Backend Integration

- Uses Supabase for backend services (authentication, database, storage)
- Type-safe database interactions with generated TypeScript types
